# v1.3.5 Final — Branding, SEO & Favicon

Cambios incluidos:

- Favicon reconstruido desde el logo como sistema completo: SVG dinámico, ICO, PNG, Apple Touch Icon, Android icons y Safari mask icon.
- Metadatos SEO revisados con español como idioma principal.
- Open Graph actualizado con la imagen profesional 1200x630.
- Twitter Cards incluidas aunque no uses X/Twitter, porque otras plataformas reutilizan estas etiquetas.
- Canonical URL oficial: https://www.ibaitudancaphoto.com/
- Sitemap público actualizado.
- Robots configurado para indexar solo páginas públicas y bloquear paneles privados.
- Schema.org reforzado para ayudar a Google a entender que la web oficial pertenece a Ibai Tudanca / Ibai Tudanca Photo.
- Headers de Vercel para evitar que Safari mantenga favicons antiguos durante el despliegue.

Después de subir a GitHub:

1. Espera al despliegue de Vercel.
2. Abre una ventana privada de Safari para comprobar el favicon.
3. En Safari, si sigue mostrando un favicon antiguo, limpia caché o espera: Safari cachea favicons de forma agresiva.
4. Entra en Google Search Console.
5. Añade la propiedad https://www.ibaitudancaphoto.com/
6. Verifica la propiedad con la opción que prefieras.
7. Envía este sitemap: https://www.ibaitudancaphoto.com/sitemap.xml
8. Usa “Inspección de URL” y solicita indexación de https://www.ibaitudancaphoto.com/
9. Añade la web en Instagram, firmas de email y perfiles profesionales para que Google la entienda como web oficial.

Importante: ningún cambio puede garantizar aparecer primero en Google al instante. Esta versión deja la web preparada técnicamente para que Google la identifique como la referencia oficial cuando la indexe.
