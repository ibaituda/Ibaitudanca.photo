# Hotfix — Checklist con origen claro

Este hotfix mejora el bloque de Checklist/Pendientes automáticos del Resumen.

Ahora cada línea indica:

- de qué sección viene el pendiente;
- qué elemento está afectado;
- qué falta exactamente;
- el color de prioridad: rojo = no iniciado/importante, amarillo = en progreso/revisar.

No requiere SQL nuevo.
