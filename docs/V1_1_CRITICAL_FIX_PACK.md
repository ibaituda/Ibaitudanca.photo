# V1.1 Critical Fix Pack

Sube a GitHub los archivos incluidos en este ZIP y ejecuta `supabase/v1_1_critical_fix.sql`.

Incluye:
- Login/preview cliente y galería corregidos.
- Logout real cliente/admin.
- Protección de acceso directo.
- Eliminación de demos iniciales en dashboard/galería.
- Calendario con colores reales y leyenda.
- Papelera con restaurar y borrar definitivamente.
- Botones de borrar cliente/galería/evento.
- Cambio de contraseña desde dashboard del cliente.
- Una sola descarga por foto.
- Subida de imágenes optimizada a WebP aprox. <=2 MB y borde corto máximo 3000 px.
